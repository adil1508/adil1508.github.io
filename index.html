Photo filtering Android app

This was a simple application that fetched some pictures over the internet, displayed them and allowed the user to filter images by assigned ratings.

This app was a first of a lot of things. First time building a fully functional android app. First time designing the entire UI. First time using Android's activity framework.

The application was written in Java. It consisted of 2 activities. One for displaying a grid of images and the other to enlarge and display that image on the entire screen.

The UI was designed mainly using the Android Studio's built-in UI GUI, with some small manual XML changes to specify the layout.

Breakout

This fun little project was a fully playable clone of the game Breakout, complete with a colorful GUI and a scoring system.

The application was written using C++ and XLibraries. This was my first attempt in dealing with User-Computer interaction. I had never before created GUIs or dealt with event loops in so direct a manner as I did when coding this.

Chamber Crawler 3000

My first experience writing a video game! I was a member of the 3 person team responsible for creating this clone of Rogue for the terminal.

The entire application was written in C++. I was also responsible for the initial software architecture before any actual code was written. Various design patterns were used in the OOP architecture to make the game easier to code while making it extensible.

PS. Our initial version was too hard to win so we also put in some secret cheat codes that only the developers know ;)

Paint Application

This application is a working clone of typical Paint applications. It only implements a subset of the features but is still powerful.
